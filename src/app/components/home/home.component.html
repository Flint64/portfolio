Brigham Young University Idaho - Ticket Management System

This is a project I worked on for Brigham Young University Idaho's Accessibility Services department as the lead developer. This project is a full stack ticket management system built in Angular allowing students, faculty, and course leads to request work to be done on a course to make sure it is ADA compliant. I was brought in to do a complete rebuild of their previous site, streamlining their workflow from various Google Docs and other spreadsheets into a one-stop shop able to handle any type of request.

Dish Recipes

This mobile site is meant to be a place where anyone can add recipes and access them at any time with no fuss, straight and to the point. This has been a long project, and the current site is actuaully version 3. I started this years ago when a friend of mine complained about having to scroll through people's life stories before getting to a recipe on the internet. So I built a prototype, which I then scrapped when I remade the site for my senior project. I later rebuilt my senior project from scratch, giving us the current Dish Recipes. This is still a work in progress, and I hope to continue improving this site over time.

The Logbook

This is an intereseting project, one that I hope to continue developing in my free time. I've always liked games and the idea of building one myself one day. I had an idea for a text-based game, and one day I figured, why not try building it? And so, we have The Logbook. Currently, it's nothing more than a combat test page, allowing you to fight in a turn-based active time battle.  I wanted a more robust text adventure, rather than your standard "read story, pick choices." I opted for a more RPG focused approach. Currently, it boasts the ability to cast spells and drink potions that can add various effects to yourself or the enemies you target. This is very much a work in progress.
